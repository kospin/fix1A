# fix1A

修正輸出文字檔尾換行和eof

## 問題

dbase資料庫經copy to輸出文字檔後在檔尾多了0D0A1A三個字元導致銀行軟體無法正常操作

## 方案

0D0A是換行字元  
1A是EOF字元  
那就把檔尾的換行和多出的一碼清掉就好了
